## What is PAXCI?

PAXCI Open Miner is a free, open-source browser-based Monero mining 
platform built for communities. No downloads, no setup, no technical 
knowledge required. Just open the site and start contributing to the 
XMR network.

Built by a small two-person team passionate about democratizing 
cryptocurrency mining for everyone — not just people with expensive 
hardware.

## Features

- ⛏️ Browser-based XMR mining via WebAssembly
- 🔒 Non-custodial wallet interface — your keys, your coins
- 📊 Real-time network statistics
- 🦊 Optimized for Firefox
- 💸 Voluntary dev donation system — fully transparent
- 🌐 No account required

## Usage

Just visit [paxci.duodevs.github.io](https://paxci.duodevs.github.io)

No installation. No signup. Open and mine.

## Firefox Recommendation

PAXCI is optimized for Firefox. Chromium-based browsers may display 
false positive security warnings due to WebAssembly compute engine 
usage. This is expected behavior and does not indicate malicious 
activity.

We recommend Firefox for the best experience.

## Dev Donations 

A small voluntary contribution is periodically collected to fund:

- Core infrastructure costs
- Developer compensation
- Security research

All transactions are visible on the Dev Donations page. We believe 
in full transparency.

## Tech Stack

- Vanilla HTML/CSS/JS
- Cloudflare Workers (proxy layer)
- [crypto-webminer](https://crypto-webminer.com) (mining engine)
- [monero-web](https://monero-web.com) (wallet interface)
